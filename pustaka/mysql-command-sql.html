| SELECT

Perintah ini untuk memilih data pada suatu tabel. 
          
SELECT [column] FROM [table];
Untuk contoh dengan data tabel ini.
Tabel: Person
Name Age Email
Ifandika 21 ifan@gmail.com
Joni 15 joni@gmail.com
Kemudian kita ingin mengambil dan menampilkan data. 
          
// Mengambil semua data
SELECT * FROM Person;

// Mengambil data tertentu
SELECT Person.name, Person.age FROM Person;
