Biner

10101 (Biner -> Desimal)
Diberikan bilangan biner 10101, maka untuk konversinya dari bilangan Biner -> Desimal, untuk langkah konversinya setiap bilangan dikali 2 dengan 2 berpangkat nilai bertambah dan dimulai dari 0 dari kanan, kemudian kita jumlahkan.
= 1×24 0×23 1×22 0×21 1×20
= 16 + 0 + 4 + 0 + 1
= 21
Jadi nilai desimalnya adalah 21

Desimal

28 (Desimal -> Biner)
Diberikan bilangan desimal 28, maka untuk konversinya dari bilangan Desimal -> Biner, untuk langkah konversinya setiap bilangan dibagi 2 higga 0 dan hasil bagi berupa 0 atau 1 yang diambil, hasil nilai pada pembagian pertama akan diletakan di paling kanan dan pembagian akhir paling kiri. Terdapat operasi matematika untuk mencari hasil dari pembagian yaitu modulo, dinotasikan mod atau %.
28 ÷ 2 = 14, sisa 0
14 ÷ 2 = 7, sisa 0
7 ÷ 2 = 3, sisa 1
3 ÷ 2 = 1, sisa 1
1 ÷ 2 = 0, sisa 1
= 11100
Jadi nilai biner dari desimal 28 adalah 11100
